# ¡SOLO HAZ ESTO! 🎯

## Paso 1: Abre pgAdmin
- Presiona `Windows`
- Escribe: **pgAdmin**
- Ábrelo

## Paso 2: Crea la base de datos
- Click derecho en **"Databases"**
- **"Create"** → **"Database"**
- Escribe: **hospital_san_rafael**
- Click **"Save"**

## Paso 3: Ejecuta el script
- Click derecho en **hospital_san_rafael**
- **"Query Tool"**
- Abre: `database/script.sql`
- Copia TODO
- Pega en Query Tool
- Presiona **F5**

## ¡LISTO! ✅

Ahora ejecuta:
```bash
mvn clean compile javafx:run
```

**¿Problemas?**

- Si tienes PostgreSQL 13 y 18, usa el **13**
